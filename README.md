## 개요
- 인스타그램을 모티브로한 소셜 서비스 프로젝트

  기술 : Java, Spring, JPA, Mysql, Docker, AWS, SSE, Elasticsearch, Web Socket, RabbitMQ
  
  목표 : 직장인을 타겟으로 피드를 올리고 서로 채팅할 수 있는 서버를 구축.
